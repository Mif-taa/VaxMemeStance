# VaxMemeStance
## 📄 CUET_SYNTHETICA@EEUCA 2026
### Gated Cross-Modal Attention with Domain-Adapted Text Encoding for Vaccine-Critical Meme Detection

**Authors:**
- Sumaiya Zaman
- Miftahul Jannat Rishta
- Shiti Chowdhury

**Venue:** Proceedings of the 9th Workshop on Event Extraction and Understanding: Challenges and Applications (EEUCA 2026)
**DOI:** [10.18653/v1/2026.eeuca-1.14](https://doi.org/10.18653/v1/2026.eeuca-1.14)

#### 🧩 Overview
This work addresses vaccine-critical meme detection using a **gated cross-modal attention architecture** that combines domain-adapted text encoding with visual features, achieving strong performance on a multimodal classification task.

#### 🏗️ Architecture
- **Text encoder:** Twitter-RoBERTa (domain-adapted for social media text)
- **Visual encoder:** CLIP ViT-L/14
- **Fusion:** Gated cross-modal attention mechanism to combine text and image representations

#### 📊 Results
🏆 **Macro F1: 0.8357** — securing **3rd place** in the shared task.
